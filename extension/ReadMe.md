# SaltVault Extension

This extension allows you to interact with your SaltVault instance.

## Features

- Login to your SaltVault instance.
- Search for passwords.
- Add new passwords.
- Autofill passwords.
- View all passwords.

## New Features

- **Settings Page**: A new settings page has been added to the extension. This page is available after you log in.
- **Change Base URL**: You can now change the base URL of your SaltVault instance from the settings page.
- **Change Session Duration**: You can now change the session duration from the settings page. The session duration is in minutes.
- **Logout & Clear Settings**: You can now log out and clear all settings (base URL and session duration) from the settings page.
